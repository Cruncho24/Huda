Where should Settings live?

Pick the entry point that feels most natural for Huda

A

Gear icon in home hero

A ⚙️ button sits in the top-right of the hero alongside the existing ? and 🔑 buttons. Tapping it opens a full-screen settings view (same pattern as the Help screen). Settings closes back to Home.

B

🔑 Account button becomes Settings

The existing account/key button already lives in the hero. Rename it to ⚙️ Settings and put account + preferences all in one place. One icon, one destination, less clutter in the hero.

C

Settings inside the Help screen

The existing ? / About Huda screen gains a Settings section at the top. User goes ? → sees preferences first, then the feature guide below. No new entry point needed.
